hypeWiper
=========
![Elgg 2.0](https://img.shields.io/badge/Elgg-2.0.x-orange.svg?style=flat-square)

## Features

- Allows users and group admins to remove or disable content in bulk
- Allows admins to remove, disable and enable content in bulk
